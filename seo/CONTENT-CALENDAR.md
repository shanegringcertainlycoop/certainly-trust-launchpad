## SEO Checklist Per Article

- [ ] Primary keyword in title, H1, URL slug, first 100 words
- [ ] Meta description with keyword (under 160 chars)
- [ ] 2-3 internal links to related content
- [ ] 1+ external link to authoritative source
- [ ] Featured image with descriptive alt text
- [ ] Schema markup (BlogPosting or NewsArticle for dispatches)
- [ ] Prerendered OG tags for social sharing
- [ ] Excerpt written for social/card display
